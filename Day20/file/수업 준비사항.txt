1. 수강동의 부탁드려요
 - mgr.kgitbank.co.kr 접속 -> 아이디찾기 -> 로그인(비밀번호 생년월일6자리)
 -> 우측 상단 나의정보 -> 좌측 출석정보 -> 자바 클릭 -> 동의 버튼 누름
 - 수강동의 하시면 그때부터 강사가 출석처리를 할 수 있습니다.
2. 카페 가입 바랍니다
 - 강사가 운영하는 개인 카페
 - http://cafe.naver.com/skyprogramming
 - 가입시 닉네임은 3월 자바 OOO 으로 해주세요(실명 확인 불가시 거절)
 - JSP 수강생은 3월 JSP OOO 으로 해주세요
3. 공유폴더에 접속하시어 설치파일을 가져가주세요
 - 공유폴더 접속은 시작메뉴 입력창에 \\192.168.10.101 라고 입력 후 엔터
 - 해당 과목의 설치파일의 내용을 복사하시면 됩니다.
4. C:\에 데이터를 저장하지 마세요
 - 학원 PC의 환경 상 C:\는 부팅시마다 초기화되도록 되어 있습니다.
 - D:\에 본인만의 폴더를 만드신 뒤 파일을 저장하시기 바랍니다.
 - 매 수업후 백업해 가셔서 혹시 모를 삭제에 대비하세요
